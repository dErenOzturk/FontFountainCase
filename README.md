# Font Fountain

## Proje Hakkında
Font Fountain, farklı fontları hızlıca inceleyip görsel olarak karşılaştırmanızı sağlayan bir uygulamadır. Seçilen font, önizleme alanında büyük boyutla gösterilir ve kullanıcı kendi metnini yazabilir. Ayrıca Glyph Rounding özelliği ile fare tekerleğiyle harfler değiştirilebilir.

## Fontlar ve Tasarım
- 50'den fazla font yerel olarak eklenmiştir ve üç kategoriye ayrılmıştır: Sans-serif, Serif, Monospace.  
- Arayüz modern ve sade tasarlanmıştır; odak font seçimine yöneliktir.

## Önizleme Metni
Varsayılan metin:  
`Fırtınalı gecede çılgın köpek, öyküsünü hızlıca yazıp şehri izledi.`  
(Tüm Türkçe karakterleri içerir ve görsellik açısından uygundur.)

## Kullanım
- Önizleme metni yazı kutusuna yazılarak değiştirilebilir.  
- Font Browser’dan font seçilebilir ve önizleme alanında görüntülenir.  
- Geçmiş fontlar Font History panelinde listelenir ve tıklanarak geri yüklenebilir.  
- Glyph Rounding: Önizleme metninde harflerin üzerine fare tekerleği ile gelerek değiştirilebilir.
- İstenen fontlar arama çubuğu yardımıyla aranabilir.

## Notlar
- Tüm fontlar ve dosyalar projede yer almaktadır.  
- Kod tamamen vanilla JS, HTML ve CSS ile yazılmıştır.
